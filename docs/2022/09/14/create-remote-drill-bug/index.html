机械动力远程钻头

Runge
2022-09-14
Create, Minecraft

提要

  • MC 1.18.2
  • Create 0.5.0d

被装配在移动结构上的钻头工作到一般半通过特殊手段移动（譬如，装配进矿车再放下来）之后会保留原来的工作状态，这意味着他可以隔空破坏方块。
